### How It Works
- Takes user input for weight (kg) and height (m)
- Computes BMI with the formula:
```
bmi = weight / (height ** 2)
```
- Displays the BMI value (rounded to 2 decimals)
- Categorizes result into:
    1. Underweight
    2. Normal weight
    3. Overweight
    4. Obesity
